Navigation View

  • What is it ?
    • Ext.Container with card layout

  • Why is it useful ?
    • Nested cards
    • Card autoremove
    • Easy to use (push/pop)

  • How does it work ?

Navigation view example

var view = Ext.create('Ext.NavigationView', {
    items: [{
        title: 'First',
        items: [{
            xtype: 'button',
            text: 'Push a new view!',
            handler: function() {
                // Use the push() method to push another view.
                // It works much like add() or setActiveItem().
                // It accepts a view instance, or you can give it a view config.
                view.push({
                    title: 'Second',
                    html: 'Second view!'
                });
            }
        }]
    }]
});

Routes (#)

  • What is it ?
    • URL anchors


  • Why is it useful ?
    • Links a part of application
    • Restores application state

  • How does it work ?

Routes example

Ext.define('MyApp.controller.Users', {
    extend: 'Ext.app.Controller',

    config: {
        routes: {
            'login': 'showLogin',
            'user/:id': 'showUserById'
        }
    },

    showLogin: function() { ... },

    showUserById: function(id) { ... }
});
